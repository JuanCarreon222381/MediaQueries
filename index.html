Ideas Principales

  • Las media queries son un modelo de css que adapta la representacion del contenido las caracteristicas de un dispositivo.
  • CSS introdujo media que es extendieron la idea de los tipos de media de CSSL en lugar de buscar un tipo de dispositivo, analizan la capacidad del dispositivo.
  • Las media queries se puede utilzar para verificar muchas cosas, ancho y alto del dispositivos, orientacion de dispositivo.
